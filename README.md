# Bank Marketing Analysis - Streamlit
Streamlit web application that predicts the likelihood of bank customers subscribing to a term deposit from personal, monetary, and marketing features.


Link to full app: https://sachinvs7-bank-marketing-streamlit-main-app-b00a9n.streamlitapp.com/

Developed with this neat documentation: https://docs.streamlit.io/
